Insert Elements
let el = document.createElement("div")
node.append( el ) //adds at the end of node (inside)
node.prepend( el ) //adds at the start of node (inside)
node.before( el ) //adds before the node (outside)
node.after( el ) //adds after the node (outside)


Delete Element

node.remove() //removes the node

Homwork Problem

appendChild() 👉 Kisi element ke andar naya child add karta hai (end me)

let div = document.querySelector("#box"); let newPara = document.createElement("p"); newPara.innerText = "New Paragraph"; div.appendChild(newPara);

removeChild() 👉 Kisi element ke andar se child ko delete karta hai

let div = document.querySelector("#box"); let para = document.querySelector("p"); div.removeChild(para);

Result: 👉 'div' tag ke andar jo 'p' tag hai wo delete ho jayega

| Method          | Kya karta hai          |
| --------------- | ---------------------- |
| `appendChild()` | child add karta hai    |
| `removeChild()` | child delete karta hai |

removeChild() me parent ko pata hona chahiye

parent.removeChild(child);
🔥 Modern shortcut:

👉 Aajkal zyada use hota hai:

para.remove(); // direct delete
Final samajh:
appendChild() → add
removeChild() → delete
remove() → easy delete
